## 打包“绿色版”实施计划

### 1. 更新项目元数据
- 修改 [package.json](file:///e:/lofi-radio/package.json) 中的 `version` 为 `1.0.2`。
- 修改 [package.json](file:///e:/lofi-radio/package.json) 中的 `description` 为 `Lofi Radio Player`。

### 2. 配置打包参数
- 在 `package.json` 的 `build` 字段中：
    - 确认 `win.icon` 为 `icon.ico`。
    - 添加 `electronLanguages: ["zh-CN", "en"]` 以移除多余语言文件。
    - 修改 `directories.output` 为 `dist_green`，方便识别。
    - 设置 `win.target` 为 `["dir"]`（仅生成文件夹，不生成安装包）。

### 3. 环境保护
- 检查并更新 [.gitignore](file:///e:/lofi-radio/.gitignore)，确保 `dist_green/` 不会被 GitHub Desktop 识别上传。

### 4. 执行打包
- 在终端运行打包命令。
- 打包完成后，目标文件将存放在 `dist_green/win-unpacked` 文件夹中。

### 5. 验证与交付
- 确认生成的 `.exe` 文件图标已更换。
- 确认 `locales` 文件夹已精简。
- 指引用户对 `win-unpacked` 文件夹进行压缩。
